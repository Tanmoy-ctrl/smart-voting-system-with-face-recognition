# Smart Voting System with Face Recognition

## Project Description
The Smart Voting System with Face Recognition is a web-based application developed using Python Flask. The system allows voters to register and cast their votes securely. Face recognition is used for voter authentication to ensure that only authorized users can vote and that each voter can vote only once.

## Features
- Voter registration system
- Face recognition authentication
- Secure voter login
- One person one vote mechanism
- Admin panel for managing voters and results
- Database storage for voter information and votes

## Technologies Used
- Python
- Flask
- MySQL / SQLite
- HTML
- CSS
- JavaScript
